**It's a simple shell to automat discovering domain process.**

* Information about Domain 
* Subdomains
* Emails

**Downloading && Usage**
```
git clone https://github.com/malbana77/Domaim
cd Domaim
chmod +x Domaim.sh
./Domain.sh
```
